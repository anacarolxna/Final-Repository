MY SUMMER

Despite all the craziness of this semester, this summer I will be working on a very interesting project in the GECo lab, the Columbia Experimental Gravity Group. In this summer research position, I will be responsible for helping with three main tasks. The first one is to help characterize the advanced LIGO timing system performance during the O3 data-taking run to ensure the quality of the gravitational wave discoveries. By measuring the timing performance, it will be possible to arrive at more accurate/precise data and advance the knowledge about GW. Another task will consist of investigating machine learning methods that can be used in detector characterization studies to improve the data quality and GW detectors. The last task is to help with multimessenger searches, which are data analysis approaches that combine various sources that together provide deeper insight into astronomical events than individual signals alone. These searches can help the understanding of elusive cosmic events, processes, and GW themselves. In this summer project, I will be required to use a lot of Python to deal with the data of the observations, so 1006 was definitely a pretty helpful class!
